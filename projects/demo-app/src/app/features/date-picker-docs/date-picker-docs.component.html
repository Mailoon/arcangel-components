Documentación — Date picker

Sustituto personalizable de input type="date": valor YYYY-MM-DD o null, calendario con calendarSize, selects nativos de mes/año, límites minDate/maxDate. El disparador comparte API con input-component (variant, shape, shapeClass, size, iconos, overrides). Teclado: Enter/Espacio en el disparador abren/cierran; Escape cierra el panel; flechas + Enter en la grilla de días.

API resumida

Input Descripción
Valor y formulario
formControl / CVA Modelo: string | null en formato ISO YYYY-MM-DD.
Disparador (como input)
variant, size, fullWidth Misma semántica que input-component.
shape rounded | rounded-md | rounded-lg | pill | square
shapeClass Si se define, sustituye el redondeo de shape (clases Tailwind).
leftIcon* / rightIcon* Clase + contenido o TemplateRef; showCalendarIcon muestra icono por defecto si no hay derecha.
placeholder, displayFormatter Texto vacío y formato de la fecha mostrada (por defecto DD/MM/AAAA).
backgroundClass, textClass, … Overrides Tailwind del wrapper del disparador.
Panel del calendario
calendarSize sm | md | lg — celdas, tipografía y padding del panel.
minDate, maxDate String ISO o Date; deshabilita días fuera del rango.
minYear, maxYear Rango del <select> de año (si no, se infiere de min/max).
firstDayOfWeek 0 = domingo, 1 = lunes (por defecto).
monthLabels, weekdayLabels Opcional; por defecto etiquetas en español.
placement bottom-start | bottom-end | top-start | top-end (preferencia inicial).
autoFlipPlacement Por defecto true: el panel del calendario elige arriba/abajo (y alineación) según espacio en pantalla. Mes y año usan dropdown-component con la misma lógica.
