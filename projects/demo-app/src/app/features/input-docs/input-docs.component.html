Documentación — Input

Misma línea visual que el botón: colores (variant), tamaño (size), ancho (fullWidth), iconos y overrides Tailwind. El borde redondeado va en shape (predefinido) o en shapeClass (cualquier clase Tailwind). Implementa ControlValueAccessor para [formControl] / formControlName.

API resumida

Input Descripción
variant primary | secondary | outline | ghost | danger | warning | success
shape rounded | rounded-md | rounded-lg | pill | square — preset de redondeo del contenedor
shapeClass Si tiene valor, sustituye el redondeo de shape (ej. rounded-2xl, rounded-t-xl).
size sm | md | lg
leftIconClass / rightIconClass Clases del <i>; leftIconContent / rightIconContent = texto (Material Icons).
leftIconTemplate / rightIconTemplate TemplateRef para iconos (p. ej. mat-icon).
backgroundClass, textClass, borderClass, hoverClass, customClass Capas Tailwind extra sobre el wrapper.
placeholder, type, inputId, name, readOnly, maxlength, autocomplete, inputMode Se pasan al <input> nativo.
disabled, ariaLabel, ariaDescribedBy, tooltip Estado y accesibilidad.
